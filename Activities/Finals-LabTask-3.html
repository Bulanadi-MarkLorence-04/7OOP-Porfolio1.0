Final Lab Task 3

Polymorphism

Problem #1: Chirp and Tweet

Create a simple program to demonstrate basic polymorphism with bird sounds.
Class Bird:
Methods:
* def make sound(self) -> None: An abstract method that represents making a sound. It doesn't have a specific implementation in the base class Bird.

Class Sparrow (extends Bird):
Methods:
* def make sound(self) -> None: Overrides the make_sound method from the base class Bird. It prints the sound "Chirp Chirp" when called.

Class Parrot (extends Bird):
Methods:
* def make sound(self) > None: Overrides the make_sound method from the base class Bird. It prints the sound "Tweet Tweet" when called.

Class BirdCage:
Methods:
* def make bird_sounds(self, birds: List) > None: Accepts a list of Bird objects as input. Iterates through the list of birds and calls the make_sound method on each bird to make its sound.

Note:
The test cases are not outputs of your main file but of a hidden test file. Create and implement the classes instructed to test your code.
Each class should be defined in its own file, with the file name following camelCase conventions (e.g., bankAccount.py).

Code for main.py File


    
from Sparrow import Sparrow
from Parrot import Parrot
from BirdCage import BirdCage

sparrow = Sparrow()
parrot = Parrot()
print(f"Sparrow says: {sparrow.make_sound()}")
print(f"Parrot says: {parrot.make_sound()}")

bird_cage = BirdCage()
birds = [sparrow, parrot]
cage_sounds = bird_cage.make_bird_sounds(birds)
print(f"Bird cage sounds: {cage_sounds}")

Code for Bird.py File


class Bird:
    def make_sound(self):
        return None

Code for Sparrow.py File


from Bird import Bird

class Sparrow(Bird):
    def make_sound(self):
        return "Chirp Chirp"

Code for Parrot.py File


from Bird import Bird

class Parrot(Bird):
    def make_sound(self):
        return "Tweet Tweet"

Code for BirdCage.py File


class BirdCage:
    def make_bird_sounds(self, birds):
        sounds = []
        for bird in birds:
            sounds.append(bird.make_sound())
        return sounds
